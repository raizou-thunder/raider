## マルコフ連鎖機能の使用方法
[MeCab](https://github.com/ikegami-yukino/mecab/releases)をインストール  
data/markov.txtにマルコフ連鎖のデータを入れます  
Config MenuにてMarkov ChainがEnabledになっていたら使用できます
